# yolo_labeling

주요 기능

1. 이미지 폴더 선택 및 라벨 폴더 선택 기능 제공
2. 마우스 드래그로 바운딩박스 생성 / 우클릭시 바운딩 박스 제거
3. 0,1,2,3 클래스 번호로 클래스 선택
4. 방향키로 이전/다음 이미지 이동
5. '현재 이미지 저장' 버튼을 통한 라벨 파일 저장 (원본 라벨 위에 덮어쓰기)


v1.2 (2025-11-24)
1. 윈도우 X버튼 클릭 시, 경고창 발생
- 윈도우 X버튼을 클릭을 하여 라벨링 프로그램을 종료할 때, "프로그램을 종료 하시겠습니까?" 문구가 뜬 후, Yes/No를 통해 프로그램을 안전하게 종료할 수 있도록 수정했습니다.

v1.1 (2025-11-18)
개선 사항

1. 이미지 전환 시 이전 라벨이 따라오는 문제 해결
- 다음 이미지로 넘어갔을 때 이전 이미지의 박스가 남아있던 버그를 수정했습니다.

2. 저장 방식 개선 – 체크된 이미지만 저장 
- '현재 이미지 저장' 버튼을 누를 경우, 해당 이미지를 실제로 수정한 경우(체크 표시된 이미지) 에만 .txt 라벨 파일이 저장되도록 수정했습니다.
