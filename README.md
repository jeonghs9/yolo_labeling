# yolo_labeling

주요 기능

1. 이미지 폴더 선택 및 라벨 폴더 선택 기능 제공
2. 마우스 드래그로 바운딩박스 생성 / 우클릭시 바운딩 박스 제거
3. 0,1,2,3 클래스 번호로 클래스 선택
4. 방향키로 이전/다음 이미지 이동
5. '저장' 버튼을 통한 라벨 파일 저장 (원본 라벨 위에 덮어쓰기)
6. Bbox 모드, Seg모드 지원

----

**v2.0 (2025-12-10)**
**1. 세그멘테이션(Seg) 라벨링 모드 추가**
   - 프로그램 실행 시, 바운딩박스(BBox)/세그먼테이션(Seg) 중 원하는 라벨링 모드를 선택할 수 있도록 기능을 추가했습니다.

**2. 마우스 클릭 또는 Enter 키로 점 추가 기능 지원**
   - 기존에는 마우스 좌클릭으로만 점을 추가할 수 있었으나, 엔터(Enter) 키만으로도 점이 추가되도록 개선하여 손 사용 피로도를 줄였습니다.
   - 이제, 마우스를 원하는 위치로 이동한 뒤 엔터만 연속으로 눌러도 편하게 폴리곤을 생성할 수 있습니다.

**3. 엔터 두 번으로 폴리곤 자동 닫기 기능 지원**
   - 엔터 키를 빠르게 두 번 누르면, 현재까지 그린 폴리곤이 자동으로 닫히고 확정됩니다.
   - 더블클릭(마우스)과 동일하게 동작하지만, 키보드만으로도 작업이 가능합니다.
